# Monetary Policy Transmission Analysis in India

## Overview

This project analyzes the effectiveness of monetary policy transmission in India using econometric techniques in Microsoft Excel. The study examines how changes in the Reserve Bank of India's (RBI) repo rate influence bank lending rates (Weighted Average Lending Rate - WALR) and inflation over time.

## Objective

- Analyze the relationship between the RBI repo rate and bank lending rates (WALR).
- Study the impact of lending rates on inflation.
- Evaluate the effectiveness of the interest rate channel of monetary policy.
- Validate findings using econometric regression and hypothesis testing.

## Tools Used

- Microsoft Excel
- Excel Data Analysis ToolPak
- Regression Analysis
- Hypothesis Testing
- Data Visualization

## Dataset

The analysis uses publicly available macroeconomic data, including:

- RBI Repo Rate
- Daily Call Money Rate
- Weighted Average Lending Rate (WALR)
- Inflation indicators

## Methodology

- Data cleaning and preparation
- Exploratory Data Analysis (EDA)
- Linear Regression Analysis
- Lagged Variable Analysis
- Hypothesis Testing
- Trend and Relationship Visualization

## Key Findings

- Identified a significant pass-through from the RBI repo rate to bank lending rates (WALR).
- Observed both immediate and lagged effects in monetary policy transmission.
- Found that lending rate changes influence inflation with a time lag.
- Regression models demonstrated satisfactory explanatory power.
- Hypothesis testing confirmed statistically significant relationships between key variables.

## Conclusion

The analysis demonstrates that changes in the RBI repo rate significantly influence bank lending rates, with transmission occurring gradually over time. The findings support the effectiveness of the interest rate channel as a monetary policy tool in India while highlighting the presence of transmission delays.
